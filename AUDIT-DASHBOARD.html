🚨 КРИТИЧЕСКИЙ БАГ — GA4 не собирает данные

GA4 показывает 0 пользователей, 0 событий, 0 сессий за последние 30 дней.

Причина: CookieConsent.tsx при повторных визитах (когда consent уже принят) возвращал null без вызова gtag('consent','update',...). Consent defaults в layout.tsx устанавливают analytics_storage: 'denied'. На каждом page load — consent оставался denied, даже для пользователей, которые нажали "Accept".

Фикс (10 марта): Добавлен вызов gtag('consent','update',...) при загрузке если stored consent существует. Data Stream показывает "Receiving traffic" (cookieless pings), но реальные данные начнут появляться только после деплоя фикса.
